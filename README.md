# 🚀 Alura Plus - Diário de Desenvolvimento 👽  

Projeto desenvolvido com **HTML** e **CSS**, baseado no protótipo disponibilizado pela [Alura](https://www.alura.com.br/).  
O objetivo foi criar uma **landing page responsiva**, garantindo uma boa experiência em diferentes dispositivos.  

🔗 **Acesse o projeto:** [lucasmb21.github.io/alura-plus](https://lucasmb21.github.io/alura-plus/)  

---
